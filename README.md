# PixelCraft Studio — GitHub Pages

This folder is ready to publish as a GitHub Pages website.

## Publish it

1. Sign in to GitHub and create a new **public** repository, for example `pixelcraft-studio`.
2. Upload every item inside this folder to the repository. Include the hidden `.github` folder.
3. Commit the upload to the `main` branch.
4. Open **Settings → Pages** in the repository.
5. Under **Build and deployment**, select **GitHub Actions** as the source.
6. Open the **Actions** tab and wait for “Deploy PixelCraft Studio to GitHub Pages” to finish.

Your website URL will normally be:

`https://YOUR-GITHUB-USERNAME.github.io/pixelcraft-studio/`

## Main pages

- `index.html` — public agency website
- `portal.html` — customer portal
- `admin.html` — admin dashboard

The portal and admin screens are front-end demonstrations. Production login, secure data, file storage, and payments require a backend.
